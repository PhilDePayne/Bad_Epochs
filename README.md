## Zależności
Do działania programu potrzebne są moduły EEGModels i mne-faster, znajdujące się w folderze 'Packages'. W przypadku EEGModels dodałem plik do zmiennej środowiskowej PYTHONPATH, mne-faster musiałem zmodyfikować na potrzeby działania programu, wersja z "oficjalnego" repozytorium nie zadziała.
